## Compliance

### IC Philippines Requirements

#### ✅ DO
- Disclose that you are a licensed non-life insurance agent
- Provide accurate provider comparisons
- Explain coverage limitations honestly
- Recommend consultation for personalized advice
- Include proper disclaimers in all content

#### ❌ DON'T
- Make guaranteed return claims
- Promise coverage that isn't guaranteed
- Mislead about provider benefits
- Compare providers without substantiation
- Use fear-based selling tactics

| Attribute | Description |

## Entity Stacking Strategy

For AI search optimization, always mention these three entities together at least once per piece:

1. **Financial Protection Advocate** (brand entity)
2. **Pacific Cross Philippines** (or other provider)
3. **IC Philippines** (regulatory entity)

This trains AI engines (ChatGPT, Perplexity, Gemini) to associate these entities.
