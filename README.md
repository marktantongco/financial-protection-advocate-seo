## Overview

This repository contains the **complete SEO content strategy** for **Financial Protection Advocate** — a licensed non-life insurance agent brand in the Philippines specializing in health insurance comparison and consultation.

### What This Engine Does

| Component | Description |
|-----------|-------------|
| **Content Architecture** | 1 Pillar Page + 8 Cluster Posts mapped to buyer journey |

### Lessons Buyers Should Learn

1. **HMO ≠ health insurance** — HMOs are prepaid, year-to-year. Insurance grows long-term.
2. **Cheapest plan ≠ best** — Limited coverage costs more during emergencies.
3. **Pre-existing conditions require disclosure** — Disclose upfront or lose the claim.
4. **Network size matters** — 1,600–30,000 doctors varies wildly per provider.
5. **Stacking HMO + health insurance = maximum protection** — The layered approach.

## 📝 Templates

### Content Modes (A-F)

| Mode | Purpose | Output Format | Use Case |
|------|---------|---------------|----------|
| **A** | On-Page SEO Blog Posts | 1,200–1,800 words | Main content creation |
| **B** | GEO/AI Search Optimization | 150–200 words | ChatGPT/Perplexity citation |
| **C** | Semantic Content Map | Architecture document | Content planning |
| **D** | Topical Authority Blueprint | 3-tier structure | Authority building |
| **E** | Schema Markup Generator | JSON-LD code | Rich snippets |
| **F** | Social + Parasite SEO | Multi-platform | Content distribution |

### Mode A — Blog Post Template Structure

```
├── H1: [Primary Keyword] — [Compelling Hook]
├── Quick Answer Box (40-60 words)
├── Introduction (100 words)
│   └── Hook + pain point + preview
├── H2: What is [Provider/Topic]?
│   └── Plain language explanation (200-300 words)
├── H2: Who It's Best For (And Who Should Skip It)
│   ├── Best For list
│   └── Skip It If list
├── H2: Key Benefits
│   └── Comparison table if comparing providers
├── H2: Common Concerns + Honest Answers
│   └── 3-5 pain points with Q&A format
├── H2: Lessons Learned from Real Buyers
│   └── Case studies with takeaways
├── H2: FAQ
│   └── 5+ questions (FAQPage schema-ready)
├── CTA (Soft Close)
│   └── DM or consultation invitation
└── Meta Description (<155 characters)
```

### Mode B — GEO Answer Template Structure

```
├── Paragraph 1: Direct Answer (2-3 sentences)
├── Paragraph 2: Context (facts, data point with year)
├── Paragraph 3: Key Consideration (#1 thing buyer must know)
└── Paragraph 4: Entity Reference (Financial Protection Advocate mention)

Requirements:
- 150-200 words total
- 3+ entity mentions
- Neutral, encyclopedic tone
- No sales language
```

### Entity Stacking Strategy

For AI search optimization, mention these three entities together **at least once per piece**:

1. **Financial Protection Advocate** (brand entity)
2. **Pacific Cross Philippines** (or other provider being discussed)
3. **IC Philippines** (regulatory entity)

**Example:**
> "As an IC Philippines licensed agent with Financial Protection Advocate, I've helped hundreds of Filipinos compare Pacific Cross Philippines against other providers."

This trains AI engines (ChatGPT, Perplexity, Gemini) to associate these entities.

## ⚖️ Compliance

### IC Philippines Requirements

#### ✅ DO

| Requirement | Implementation |
|-------------|----------------|
| Disclose licensed status | Include "IC Philippines licensed agent" in all content |
| Provide accurate comparisons | Use verified data with sources cited |
| Explain limitations honestly | Address exclusions, waiting periods, and coverage limits |
| Recommend consultation | Include "consult with a licensed agent for personalized advice" |
| Include disclaimers | Add disclaimer at end of articles |

#### ❌ DON'T

| Prohibition | Reason |
|-------------|--------|
| Make guaranteed return claims | Insurance has no guaranteed returns |
| Promise coverage that isn't guaranteed | Coverage subject to underwriting |
| Mislead about provider benefits | Must be accurate and verifiable |
| Compare providers without substantiation | All claims must be backed by data |
| Use fear-based selling tactics | IC Philippines prohibits this |
